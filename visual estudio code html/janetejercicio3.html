World wide web

En informatica,la strong> (www) o red informatica mundial comunmente conocida como la wed, es un sistema de distribucion de documentos hipertexto o hipermedios interconectados y accesible via internet.con un navegador un usuario visualiza sitios web conpuestos de paguina web que puede contener texto,imagenes,videos u otros contenidos multimedia y navega a traves de esta paguina usando hiperenlaces.

Historia

la web se desarrollo entre marzo de 1989 y diciembre de 1990 por el ingles Tim Berners-lee con la ayuda del belga Robert Cailliau mientras trabajaban en el CERN en Ginebra, Suiza, y publicado en 1992.

Desde entonces,Berners-Lee ha jugado un papel activo guiando el desarrollo de estandares web (como los lenguajes de marcado con los que se crean las paginas web), y en los ultimos años ah abogado por su vision de una web semantica.
